# 🌸 Mensaje Rosas Día de la Mujer

Una experiencia interactiva y emotiva para celebrar el Día Internacional de la Mujer. Crea hermosas flores digitales mientras lees un mensaje especial.
<img width="1703" height="951" alt="image" src="https://github.com/user-attachments/assets/53c084d9-9708-4b47-8e33-28d38a1ced9f" />

## ✨ Características

- 🎨 **Flores interactivas**: Crea flores de colores arrastrando el cursor o tocando la pantalla
- 📱 **Diseño responsive**: Funciona perfectamente en desktop, tablets y móviles
- 💫 **Animaciones suaves**: Transiciones elegantes y efectos visuales profesionales
- 🎭 **Interfaz moderna**: Diseño glassmorphism con tipografía elegante
- 🌈 **Colores vibrantes**: Paleta de colores en tonos rosados y morados

## 🚀 Demo

Visita la página en vivo: [Ver Demo](https://andres-hincapie-hrz.github.io/Mensaje-Rosas-Dia-de-la-Mujer-/)

## 🎮 Cómo usar

### En Desktop
- Mantén clic presionado y arrastra por la pantalla para crear flores

### En Móviles
- Toca y arrastra por cualquier parte de la pantalla para crear flores

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (con animaciones y efectos modernos)
- JavaScript (Vanilla JS)
- SVG para gráficos vectoriales
- Google Fonts (Playfair Display & Poppins)

## 📦 Instalación

1. Clona este repositorio:
```bash
git clone https://github.com/Andres-Hincapie-HRZ/Mensaje-Rosas-Dia-de-la-Mujer.git
```

2. Navega al directorio del proyecto:
```bash
cd Mensaje-Rosas-Dia-de-la-Mujer
```

3. Abre `index.html` en tu navegador favorito

## 📁 Estructura del proyecto

```
Mensaje-Rosas-Dia-de-la-Mujer/
│
├── index.html          # Estructura HTML principal
├── style.css           # Estilos y diseño responsive
├── script.js           # Lógica de interacción y animaciones
└── README.md           # Este archivo
```

## 🎨 Personalización

Puedes personalizar fácilmente:

- **Colores de las flores**: Edita el array `colors` en `script.js`
- **Mensaje**: Modifica el contenido en `index.html`
- **Estilos**: Ajusta los colores y fuentes en `style.css`
